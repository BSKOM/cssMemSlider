CSS Memas Slider

This type of slider is an incomplete prototype of the original type of sliders developed by me with the following features: property order, radio input type tags, more than one label tag per input, as well as another key (not implemented in this solution) component that should significantly simplify and expand capabilities.

It is assumed that upon completion of development, the slider on css will be widely used because it has a number of advantages. In last year the statistics of the use of slides are incredible - many times higher. For example, it is common practice to use sliders to style blogs.
